# Catálogo de Elementos

Este proyecto es un programa en JavaScript que gestiona un catálogo de cursos de idiomas.

Se aplican métodos de objetos y arrays para manipular y analizar la información.

## Cómo ejecutar

```bash
node script.js
```

## Funcionalidades

* Visualización de elementos
* Inspección con Object.entries y for...in
* Filtrado y búsqueda de datos
* Cálculo de estadísticas
* Ordenamiento de elementos
* Uso de spread operator

## Tecnologías

* JavaScript
* Node.js
