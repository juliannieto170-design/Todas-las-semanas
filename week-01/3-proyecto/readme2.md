# Proyecto Semana 1: Ficha de Presentación de Dominio

## 🎯 Objetivo
Aprender a usar **JavaScript en Node.js** para mostrar una ficha de presentación de un curso en la consola.

## 📋 Qué trabajé
- Organizar información clara en secciones  
- Usar `console.log()` para mostrar datos  
- Manejar tipos de datos: `string`, `number`, `boolean`  
- Hacer cálculos simples (descuento)  
- Separar la salida con encabezados y líneas para que sea legible

## 🧩 Entidad usada
**Curso de idioma**  
- Nombre: Curso de Inglés Básico  
- Código: EL-101  
- Categoría: Idiomas – Nivel principiante  
- Estado: Activo
