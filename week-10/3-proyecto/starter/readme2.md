# 📦 Proyecto Integrador — Etapa 0

Aplicación en JavaScript que simula la gestión de un inventario digital mediante operaciones CRUD, análisis de datos y generación de reportes en consola.

## 🚀 Funcionalidades

* Agregar elementos al inventario
* Buscar por ID
* Filtrar por categoría o estado (activo/inactivo)
* Actualizar elementos de forma inmutable
* Calcular estadísticas (mínimo, máximo, promedio y total)
* Generar un reporte completo en consola

## 🛠️ Tecnologías

* JavaScript (ES2023)
* Node.js

## ▶️ Ejecución

```bash
node 3-proyecto/starter/app.js
```

## 📌 Notas

Proyecto enfocado en el uso de arrays, funciones, métodos como `map`, `filter`, `find` y buenas prácticas en JavaScript moderno.
