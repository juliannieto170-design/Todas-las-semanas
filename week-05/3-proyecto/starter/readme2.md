# Clasificador de Cursos

Este proyecto es un programa en JavaScript que clasifica un curso de una plataforma de idiomas según su progreso, estado y tipo.

Se usan condicionales como **if/else, ternario y switch**, además de operadores como **?? y ?.**.

## Cómo usar

Abrir la terminal y ejecutar:

```
node script.js
```

## Qué hace

Muestra en consola una ficha con información del curso, como:

* Nombre
* Estado
* Clasificación
* Tipo

## Tecnologías

* JavaScript
* Node.js
