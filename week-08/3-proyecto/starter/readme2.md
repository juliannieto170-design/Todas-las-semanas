# Gestión de Inventario

Este proyecto es un programa en JavaScript que simula la gestión de cursos de una plataforma de aprendizaje de idiomas.

Se trabajan operaciones sobre un inventario usando arrays y funciones.

## Cómo usar

Abrir la terminal y ejecutar:

```id="p9x2la"
node script.js
```

## Qué hace

Muestra en consola:

* Lista de cursos
* Agregar y eliminar cursos
* Búsqueda de cursos
* Filtrar cursos activos
* Transformación de datos con map

## Tecnologías

* JavaScript
* Node.js
