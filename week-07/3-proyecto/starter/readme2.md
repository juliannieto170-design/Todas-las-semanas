# Librería de Funciones

Este proyecto es un programa en JavaScript que simula una plataforma de aprendizaje de idiomas usando funciones.

Se crean funciones para formatear datos, validar elementos y realizar cálculos sobre los cursos.

## Cómo usar

Abrir la terminal y ejecutar:

```id="d2k9s1"
node script.js
```

## Qué hace

Muestra en consola:

* Lista de cursos
* Cantidad de cursos activos
* Total del progreso
* Información formateada usando funciones

## Tecnologías

* JavaScript
* Node.js
