# Reporte de Cursos

Este proyecto es un programa en JavaScript que genera un reporte de cursos de una plataforma de idiomas usando bucles.

Se usan estructuras como **for...of y for clásico** para recorrer datos, contar elementos, calcular totales, promedios y encontrar valores máximos y mínimos.

## Cómo usar

Abrir la terminal y ejecutar:

```id="z8xk2p"
node script.js
```

## Qué hace

Muestra en consola:

* Lista de cursos
* Conteo por categoría
* Total y promedio de progreso
* Curso con mayor y menor progreso
* Un reporte indicando si cada curso está sobre o bajo el promedio

## Tecnologías

* JavaScript
* Node.js
